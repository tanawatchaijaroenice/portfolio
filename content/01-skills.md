# Skills — ทักษะเทคนิค

> ใช้สำหรับ section: **expertise**
> หลักการ: เรียงตามความถนัดจริง — Angular คือ core strength, fullstack ระบุตามจริง (ไม่โม้)

## Core — เชี่ยวชาญจริง (ใช้ทำงาน production ทุกวัน)

### Angular (จุดแข็งหลัก / Primary strength)
- RxJS / Observables — จัดการ async, operators, state ด้วย stream
- Signals (Angular ใหม่ 16+) — reactive primitives
- NgRx / State Management — global state ด้วย store pattern
- Reactive Forms — validation, dynamic forms, ฟอร์มที่ซับซ้อน
- Component architecture, UI logic, drag & drop, table/list, calendar

### TypeScript
- Typing, generics, interfaces — ใช้แบบ strict

### Styling
- SCSS
- Tailwind CSS

## Secondary — ใช้ได้คล่อง (React ecosystem)

### React / Next.js
- App Router + Server Components (Next.js 13+)
- API Routes / Server Actions
- React hooks พื้นฐาน (useState, useEffect, custom hooks)
- จัดการ cart / checkout / session storage

## Tooling & Workflow
- Git (version control)
- Git Workflow (branching, merging, pull requests, collaboration)

## Backend / Data — ระบุตามจริง (Frontend-leaning fullstack)

> **สำคัญ:** นำเสนอแบบซื่อสัตย์ — เข้าใจ concept และทำงานร่วมกับ backend ได้ดี แต่ไม่ใช่ backend เต็มตัว

- REST API (พื้นฐาน) — เรียก / ต่อ REST API, HTTP client, interceptors, error handling
- Node.js (พื้นฐาน) — เขียน endpoint และ API logic ง่ายๆ
- Prisma (ORM จัดการ Database query & migrations)
- เข้าใจ Database / SQL พื้นฐาน — query, schema, relations
- เคยลองเรียก LLM API
